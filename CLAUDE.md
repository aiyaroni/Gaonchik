# CLAUDE.md — Gaonchik Source of Truth
> קובץ זה הוא האמת היחידה לגבי סגנון, קול, ומבנה שאלות בפרויקט גאונצ'יק.
> כל שאלה, טקסט ממשק, ורכיב חדש חייבים לעמוד בהנחיות אלו.

---

## 1. הקול של גאונצ'יק (Tone & Voice)

- **מעצים, סקרן, חברותי.** הילד הוא גיבור הסיפור — לא תלמיד שנבחן.
- **שופי הינשוף** הוא הדמות המלווה הקבועה:
  - `tiny` / `junior` — שופי כגיבור הסיפור ("שופי גילה...", "שופי ניסה...")
  - `mid` / `senior` — שופי כ"ראש המעבדה" ("שופי ראש המעבדה מציג...", "שופי מאתגר אתכם:")
- **אין מורה, אין ציונים, אין כישלון** — רק "כמעט, בוא ננסה שוב".

## 2. חוקי העברית

### מותר ואסור

| במקום | השתמש ב |
|-------|---------|
| כיצד | איך |
| וחש / הרגיש | והרגיש |
| הינו / הנו | הוא |
| נכפפת | משנה כיוון / מתכופפת |
| בעת | כש / בזמן ש |
| על מנת | כדי |
| בשל | בגלל |

### שפה מגדרית — חובת שימוש ב-`t()`

כל פנייה ישירה לשחקן (בטקסט שאלה, ברמז, בהודעות ממשק) **חייבת** להשתמש ב-`t()` מ-`core.js`.

```js
// דוגמה נכונה בשאלה:
hint: () => `כוח המשיכה מאט את הכדור. ${t('tryAgain')}!`

// דוגמה נכונה בממשק:
t('chooseAnswer')   // → 'בחר תשובה ואז אשר' / 'בחרי תשובה ואז אשרי'
```

מפתחות `t()` קיימים ב-`core.js → STRINGS`:
`greeting`, `wellDone`, `tryAgain`, `continue_`, `correct`, `almost`, `chooseAnswer`, `letsGo`

### ש' הזיקה
- נכון: "לפני שהוא נופל", "אחרי שגמר"
- לא נכון: "לפני נפילתו", "עם סיומו"

## 3. הגדרות שכבות גיל (Layers)

### Tiny — גן חובה / כיתה א (גיל 4–6)
- **3 תשובות בלבד** (לא 4)
- מסיחים: מצחיקים, בלתי אפשריים, חושיים ("הפך לפרפר 🦋", "עף לירח 🌙")
- שפה: משפטים קצרים, פעלים פשוטים, אין מונחים
- אימוג'ים: **חובה** בכל תשובה ובטקסט השאלה
- שופי: גיבור הסיפור

### Junior — כיתות ב–ד (גיל 7–9)
- **4 תשובות**
- מסיח אחד מצחיק/אבסורדי, השאר סבירים
- מושגי יסוד: שמות תופעות, קשרי סיבה-ותוצאה פשוטים
- אימוג'ים: **מומלץ** בשאלה, אופציונלי בתשובות
- שופי: גיבור הסיפור

### Mid — כיתות ה–ו (גיל 10–12)
- **4 תשובות**, כולן אמינות
- חשיבה מדעית: הסברים מנגנוניים, "מה גורם ל..."
- שפה בוגרת אך עדיין לא אקדמית
- אימוג'ים: **בשאלה בלבד** אם תורמים להבנה
- שופי: ראש המעבדה

### Senior — כיתות ז–ח (גיל 13–14)
- **4 תשובות**, כולן אמינות מדעית
- נושאים: חוקי ניוטון, אנרגיה, כבידה, גלים, חשמל בסיסי
- **מותר:** חישובי F=ma, אנרגיה קינטית/פוטנציאלית, תדר/אמפליטודה
- **אסור:** חוק סנל, מעגלי נגדים מסובכים, טריגונומטריה, גזירות
- אימוג'ים: **לא** בשאלות Senior
- שופי: ראש המעבדה

## 4. מבנה שאלה — פורמט טכני (`questions.js`)

```js
{
  id:       'category_layer_topic_nn',   // דוגמה: 'mech_tiny_g01'
  subject:  'science',
  category: 'mech' | 'waves' | 'matter_energy' | 'space' | 'electric',
  topic:    'gravity' | 'motion' | 'light' | 'sound' | 'magnet' | 'energy' | 'matter' | 'space' | 'electric',
  layer:    'tiny' | 'junior' | 'mid' | 'senior',
  q:        () => string,   // פונקציה — יכולה לקרוא ל-t()
  choices:  string[3|4],   // 3 ל-tiny, 4 לשאר
  correct:  number,         // אינדקס 0-based
  hint:     () => string,   // פונקציה — חייבת לסיים ב-t('tryAgain') אם פונה לשחקן
}
```

**Helper נדרש בקוד האפליקציה:**
```js
function resolveText(field) {
  return typeof field === 'function' ? field() : (field || '');
}
```

## 5. קטגוריות ושמות ידידותיים לממשק

| `category` (טכני) | שם בממשק | `emoji` |
|-------------------|----------|---------|
| `mech`            | איך דברים זזים? | 🏃 |
| `waves`           | קסם האור והצלילים | 💡 |
| `matter_energy`   | מה יש בתוך הדברים? | 🧪 |
| `space`           | כוכבים ועולמות רחוקים | 🪐 |
| `electric`        | כוח החשמל והמגנט | ⚡ |

**חשוב:** שמות ממשק ידידותיים לילדים. המונחים הטכניים (mech, waves וכו') נשמרים ב-JS בלבד.

## 6. עיצוב (Design Language)

- **סגנון:** וקטורי ונקי — Duolingo-inspired. לא photorealistic, לא clipart ישן.
- **צבעים:** הגרדיאנט הבסיסי `from-blue-100 via-purple-50 to-green-100` — שמור עליו בכל המסכים.
- **כרטיסי תשובה:** `choice-card` — padding רחב, border-radius גדול, hover ברור.
- **אנימציות:** `confettiBurst` על תשובה נכונה. `playSuccessSound` / `playErrorSound` מ-`core.js`.
- **נגישות:** כל אלמנט אינטראקטיבי עם `aria-label` בעברית. תמיכה ב-Tab navigation.

---

## 7. מה לא לעשות (Anti-patterns)

- ❌ שאלות שמביישות טעות ("טעית! התשובה הנכונה היא...")
- ❌ מונחים אקדמיים ב-Tiny/Junior ללא הסבר
- ❌ מסיחים אמינים מדי ב-Tiny (ילדים יתבלבלו)
- ❌ יותר מ-4 תשובות בכל שכבה
- ❌ טקסט מגדרי קשיח בלי `t()` — תמיד דרך המערכת
- ❌ חישובים עם טריגונומטריה ב-Senior
- ❌ שאלות על מגנטים בקטגוריית `mech` — הן שייכות ל-`electric`
- ❌ מהירות האור: תמיד "300,000 קמ"ש (קילומטרים **בשנייה**)"
